# Currículo ATS em HTML5 + Bootstrap

Este repositório contém um currículo profissional otimizado para ATS (Applicant Tracking Systems), utilizando HTML5 puro e Bootstrap 5 para garantir responsividade, clareza e fácil manutenção.

## Principais Características

- **Layout responsivo:** Estruturado com o grid do Bootstrap, se adapta a diferentes dispositivos e impressão.
- **Seções completas:** Objetivo, Resumo Profissional, Habilidades Técnicas, Experiência, Formação, Certificações, Idiomas e Informações Adicionais.
- **Foco em boas práticas de código:** Classes legíveis, uso extensivo de utilitários Bootstrap e mínima customização CSS.
- **Acessibilidade:** Uso de tags semânticas (header, section, ul, li) e contraste adequado.
- **Pronto para impressão:** Estilos otimizados para papel A4 com media queries específicas.
- **Tecnologias destacadas:** HTML, CSS, JavaScript, TypeScript, React, Next.js, Node.js, PHP, MySQL, Docker.

## Visualização

Abra `index.html` no navegador de sua preferência. O currículo está pronto para impressão diretamente pelo navegador.

## Como usar este modelo

1. **Fork/clone** este repositório.
2. Edite o arquivo `index.html` com seus dados.
3. Visualize em tela ou imprima para PDF.

## Estrutura do projeto

- `index.html` — Arquivo principal do currículo com toda a marcação e estilos inline necessários.
- `.cursor/rules.mdc` — Define as diretrizes para desenvolvimento contínuo com foco em Bootstrap, clareza e acessibilidade.
- `readme.md` — Orientações e visão geral do projeto.
